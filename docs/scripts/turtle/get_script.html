Get ScriptÂ¶

Downloads/updates a single script or API. You generally won’t need to bother with running this; Get All runs this for you for all scripts and APIs, and doesn’t require any arguments.

> bin/get_script <PATH> <HASH>

You must pass the path (eg bin/scriptname) and the eight-character pastebin.com hash associated with the script.
